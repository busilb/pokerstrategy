■ 翻前位置策略

前位 UTG / UTG+1 只打精品

  • 范围：TT+ AKo AKs AQs（仅6%）
  • 尺寸：Open 3BB（60积分）
  • 原则：宁可一圈不进，不打边缘牌
  • 禁忌：不用小对子前位limp，不用suited connectors，JQo/89o这种绝对不打！
  • 翻后：中了顶对直接bet拿value，不要check-call三条街

中位 MP / HJ 稳中求进

  • 范围：88+ AQo+ KQs AJs（约10%）
  • 尺寸：Open 3BB，有人limp加到5BB（100积分）
  • 面对3bet：JJ+/AK继续，其他fold
  • 翻后：中了就续火（翻牌bet转牌bet），没中就放弃

后位 CO / BTN 你的印钞机

  • 范围：66+ ATo+ KJo+ 76s+（18-25%）
  • 偷盲：2.5BB（50积分），前面都fold就更宽
  • BTN特权：open任何broadway + Axs + suited connectors
  • 原则：后位是利润来源，大胆偷盲。被3bet就fold边缘牌
  • 优势：有位置 = 最后行动 = 信息最多 = 打得更准

盲位 SB / BB 防守有度

  • 面对小raise：top40%防守（对子、broadway、suited连子）
  • 面对3bet：不冷call，4bet或fold
  • SB：最差位置，宁可多fold
  • BB：已投1BB可稍宽call，但翻后没位置要谨慎
  • 关键：被偷盲不丢人，硬撑才亏钱

■ 进阶理论 · 用数学打牌

一、纳什均衡（Nash Equilibrium）

什么是纳什均衡？

简单说：一种不可被exploit的打法。当你按纳什均衡策略打时，无论对手怎么调整，长期都不能从你身上赚到更多钱。

类比：就像石头剪刀布，各出1/3的概率就是纳什均衡。对手无论怎么猜，长期赢率都是50%。

纳什均衡在德扑中的应用：

1. 翻前push/fold（短码专用）
当筹码小于等于15BB时，纳什均衡告诉你：哪些牌该全推，哪些该弃。不需要考虑翻后。

2. Bluff频率平衡
河牌下注时，纳什均衡要求bluff比例约等于 下注尺寸/(下注+底池)。
比如pot bet时，bluff比例应该约33%。

3. 面对下注的call频率
对手下注时，为了不被bluff exploit，你需要call的频率约等于 底池/(底池+下注)。
对手pot bet时，你需要call约50%的range。

布斯的应用建议：

你目前不需要严格按GTO打（朋友局对手不平衡），但需要理解一个原则：
当你不确定对手是什么类型时，按纳什均衡打不会犯大错。
当你读到对手有明显倾向时，偏离均衡去exploit他。

二、赢率 / 胜率 / Equity

Equity（权益/胜率）= 你的手牌在当前局面下赢得底池的概率。

Equity = 你能赢的次数 / 总次数 x 100%
翻牌圈的equity会随转牌、河牌变化

三、Pot Odds（底池赔率）

Pot Odds = 需要跟注的金额 / (底池 + 需要跟注的金额)
当你的赢率 > Pot Odds 时，跟注就是+EV的

实战快算法：4-2法则

翻牌圈outs数 x 4 约等于到河牌的胜率%
转牌圈outs数 x 2 约等于到河牌的胜率%

举例：你有同花听牌（9个outs）
翻牌：9 x 4 = 36%（实际35%）
转牌：9 x 2 = 18%（实际19.6%）

常见outs数：
同花听牌 = 9 outs = 翻牌36% / 转牌18%
两头顺听 = 8 outs = 翻牌32% / 转牌16%
卡顺听牌 = 4 outs = 翻牌16% / 转牌8%
一对变两对/暗三 = 5 outs = 翻牌20% / 转牌10%

四、EV（期望值）

EV = (赢的概率 x 赢的金额) - (输的概率 x 输的金额)
EV > 0 就是正确决策，长期执行必然盈利

EV计算实例

场景：底池800，对手河牌all-in 400，你有同花听牌（9 outs）

你需要跟400，底池总共800+400+400=1600
Pot Odds = 400/1600 = 25%
你的胜率（转牌圈）= 9x2 = 18%

18% < 25% = fold是正确的（-EV call）

但如果是翻牌圈（还有两张牌要来）：
你的胜率 = 9x4 = 36%
36% > 25% = call是正确的（+EV call）

五、SPR（筹码底池比）

SPR = 有效筹码 / 翻牌圈底池
SPR决定了翻后打法的弹药量
SPR范围含义策略
SPR 4以下短码对抗顶对就能打光，别怕commit。一对+好踢脚就是nuts级别
SPR 4-10中码对抗两对以上打光，顶对需谨慎。set mining开始有价值
SPR 10以上深码对抗需要强牌才能打光。更多post-flop技巧空间
你今天的SPR：
起手1000筹码，Open 3BB(60)被call，翻牌底池约150。
SPR = 940/150 约等于 6.3 = 中码对抗
意味着：两对以上大胆打光，纯顶对需要控池思维。

六、Fold Equity（弃牌权益）

实际赢率 = Equity + Fold Equity
你的牌不够好，但对手会fold = 你的实际赢率比手牌赢率高

为什么要理解Fold Equity？

你拿着同花听牌（裸equity 35%），但如果你全推，对手有30%的概率弃牌：

实际赢率 = 30%(对手fold你直接赢) + 70% x 35%(被call但你中花) = 30% + 24.5% = 54.5%

这就是为什么听牌全推在50BB深度下经常是+EV的。不只是靠中牌赢，还靠对手弃牌赢。

布斯适用：你目前太少利用fold equity（紧弱的核心问题）。当你转牌续火时，本质就是在积累fold equity。

  • Fold Equity — 对手弃牌带来的额外赢率
  • Range — 手牌范围（对手可能拿到的所有牌的集合）

对手看你像一本打开的书：入池宽 + 好牌不放 = 印钞机（给对手的）。

弱点清单

  • 🔴 入池太宽(VPIP 30%) = 打了太多垃圾牌，前位89o/JQo这种别进
  • 🔴 好牌上头不能fold = AA/AK拿到就觉得天下无敌，被raise还推all-in
  • 翻牌不中只偷一枪 = 对手摸清pattern：你cbet了不中就停
  • 顶对面对大注不fold = 辛苦等来好牌不舍得放的心理
  • 转牌check河牌弱 = 对手读到转牌check=放弃，河牌随便偷
  • 河牌非nuts还raise = 情绪覆盖理性，给对手送钱
  • AKo接翻前all-in = 对手推all-in的range是AA/KK/QQ，AK胜率不够

经典亏钱手牌复盘

手牌1：AA转牌被raise → 推all-in → 输 好牌上头
AA open → 翻牌cbet → 转牌bet 2/3 pot → 对手raise → 推all-in
错误：AA只是一对！对手转牌raise你两条街的bet = 他有两对/set以上。正确做法：call看河牌，不要推。推all-in只会被更好的牌call，更差的牌fold。
正确：转牌被raise → call → 河牌面没变差再决定
手牌2：AKo接翻前all-in → 对手KK → 输 不该接
翻前对手推all-in → AKo call → 对手翻KK → 输
错误：朋友局翻前all-in的range = AA/KK/QQ。AKo vs 这个range胜率约25-30%，远不够call。而且你的K被对手的KK block了。
正确：面对翻前all-in → AKo直接fold。只有AA才稳接，KK可以考虑。
手牌3：JJ中set输顺子
翻牌小小小（连张）= 转牌听顺到了 = 河牌J中set = 对手all-in = 我call = 对手翻顺子
错误：连张面set不是nuts，对手all-in = 顺子。应该fold。
手牌4：AK顶对河牌raise输两对
AK前位open = 被3bet = call = 翻牌有A = 三条街check-call = 河牌raise = 对手AJ两对
错误：1.河牌raise（非nuts不raise）2.三条街被动最后一街突然上头 3.应该翻牌就主动bet
手牌5（打得好）：JQo转牌果断fold 放下到位
JQo open → 翻牌T-T-8 check → 转牌A bet 1/3 pot → 对手raise → fold
正确：只有卡顺听牌(4 outs=8%)，面对raise赔率完全不够。放下原则执行到位！

改善方向（优先级排序）

  • 🔴 P0 - 收紧入池：VPIP从30%降到20%以下。前位(UTG/MP)只打TT+/AK，中位加88+/AQ，后位最宽也别超25%
  • 🔴 P0 - AA/AK不上头：AA被转牌raise = call不推。AKo面对翻前all-in = fold。一对就是一对，不是nuts
  • 🟡 P1 - AK翻前不接all-in：50BB深度，朋友局对手推all-in = AA/KK/QQ，AKo胜率不够
  • 🟡 P1 - 河牌顶对不加注：只call或fold，非nuts不raise
  • 🟢 P2 - 翻牌公对面可以cbet：公对面对手也很难中，fold equity高
  • 🟢 P2 - 续火改进：翻牌cbet后如果面没变差，转牌再打一枪
  • 思考层级：我读他有什么 → 他在读我什么 → 我该怎么表演

三、翻后决策树

翻牌发出来了，你中了什么？
├── 坚果牌（两对/set/顺/花）
│   └── bet-bet-bet 三条街快打，别慢打
├── 顶对好踢（AK中A / KK中K）
│   ├── 翻牌 bet 2/3 pot
│   ├── 转牌 bet 2/3 pot（续火!）
│   └── 河牌 check-call（绝不raise!）
├── 中对 / 弱顶对
│   ├── check-call 一条街
│   └── 面对大注(pot+) → FOLD
├── 听牌（顺听/花听）
│   ├── 有位置 + 对手check → bet semi-bluff
│   ├── pot odds够 → call
│   └── 50BB大draw → 可以翻牌全推(fold equity)
└── 空气牌
    ├── Cbet 一枪（2/3 pot）
    ├── 被call → 转牌面没变差再打一枪
    └── 不要三枪bluff!

四、必须FOLD检查清单

以下场景默认FOLD，不要犹豫：
☐ 河牌对手bet pot+ → 你只有一对 → FOLD
☐ 三花面/四连面 → 你没花没顺 → FOLD
☐ 转牌对手raise你的bet → 你没有两对以上 → FOLD
☐ 对手三条街大注 → 你只有顶对 → FOLD
☐ Set面对连张board对手all-in → 可以FOLD
☐ AKo面对翻前all-in → FOLD
☐ AA被转牌raise → 只call不推
